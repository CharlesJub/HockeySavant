About the Site

This site is made in the style of the popular baseball resource Baseballsavant but with Hockey data. The goal for this site was to make advanced analytics more digestible. On the site, you'll find a wide range of hockey data, including player percentile metrics, goal locations, and highlight clips. It was inspired by Neil Pierre-Louis's analytics site. The site was created using Flask in python and is hosted using PythonAnywhere.

About Data

The data used for this site was collected by me using the NHL's API (depreciated endpoints). The code for scraping can be found here. All the calculations for the data presented on the site was also done by me. The code for the calculations are not on the GitHub, but if you have any questions you can reach out to me, and I'll be happy to share. The data is stored in a database using SQLite.
